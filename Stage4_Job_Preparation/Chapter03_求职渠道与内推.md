## 3.3 内推

### 为什么内推重要？

> **内推 = 简历优先处理 + 面试通过率翻倍**

| 维度 | 无内推 | 有内推 |
|-----|-------|-------|
| 简历处理 | 进入池子排队 | 直接到面试官 |
| 反馈速度 | 1-2周 | 3-5天 |
| 通过率 | ~5% | ~15% |

### 内推渠道

```
优先级（从高到低）：
1. 导师/教授推荐（大厂有导师项目）
2. 学长学姐内推（最直接）
3. 牛客网内推板块（实时更新）
4. 脉脉内推（私信 HR）
5. GitHub 项目被公司的人关注
6. 参加公司 Hackathon
```

### 牛客网内推

```
1. 注册牛客网
2. 进入"职位" → "内推"
3. 搜索公司名 + 岗位
4. 找到内推人，发私信
5. 简历发给内推人
```

### 找内推人的方法

```
1. 脉脉：搜索"公司名 + 员工"
2. LinkedIn：搜索"公司名 + 工程师"
3. GitHub：搜索公司仓库的 contributor
4. 知乎：搜索"公司名 + 招聘"
5. 朋友的朋友（社交网络）
```

### 内推话术

```
Hi [名字]学长/学姐，

您好！我是 [学校] [专业] 的 [年级] 学生，
看到您在 [公司] 工作，非常仰慕！

我想应聘 [岗位]，我的简历：xxx
GitHub：xxx
项目经验：xxx

请问方便帮我内推一下吗？非常感谢！
```

## 3.5 面试流程

### 标准流程

```
一面（技术基础）45-60min
    ↓（通过）
二面（项目经验）45-60min
    ↓（通过）
三面（系统设计/HR）30-45min
    ↓（通过）
Offer
```

### 一面：技术基础

```
考察内容：
- 数据结构与算法
- 计算机基础（OS/网络）
- 游戏开发专业知识

常见问题：
1. "介绍一下你做的最多的项目"
2. "C++ 虚函数的实现原理"
3. "Unity 的生命周期函数"
4. "写一个二分查找"
```

### 二面：项目经验

```
考察内容：
- 项目细节（做了什么，怎么做的）
- 问题解决能力
- 系统设计能力

STAR 法则回答：
Situation → Task → Action → Result
每个项目准备 2-3 个亮点故事
```

### 三面：HR/系统设计

```
HR 面：
- 离职原因
- 职业规划
- 期望薪资
- 有什么问题要问

系统设计（技术岗）：
- "设计一个斗地主游戏"
- "设计一个排行榜系统"
- "如何实现帧同步"
```

## 3.6 谈薪资

### 市场价位（2024 届参考）

| 维度 | 初级(0-2年) | 中级(3-5年) | 高级(5年+) |
|-----|------------|------------|-----------|
| 一线大厂 | 25-35W | 40-60W | 80W+ |
| 中厂 | 15-25W | 25-40W | 50W+ |
| 小厂/独立 | 10-18W | 18-30W | 40W+ |

### 谈薪资技巧

```
1. 了解市场价位（Offershow/脉脉）
2. 先让对方出价（"请问这个岗位的薪资范围是？")
3. 说出期望值（稍高于实际期望，留谈判空间）
4. 关注打包（Base + 期权 + 签字费 + 奖金）
5. 不要只盯着钱（成长空间/项目/技术栈同样重要）
```

### 打包构成

```
Total Package = Base × 12 + 年终奖 + 期权/股票 + 签字费

常见年终奖：
- 大厂：4-6个月
- 中厂：2-4个月
- 小厂：1-3个月（看项目）
```

## 3.7 入职准备

### 离职前

```
1. 提前 30 天书面通知
2. 交接工作（不要删库跑路）
3. 收集证明（离职证明/薪资证明/社保记录）
4. 背调准备（确保简历真实）
```

### 背景调查

```
通常查：
- 学历真实性
- 工作经历真实性
- 薪资真实性

注意事项：
- 简历不要造假（学历/工作时间）
- 同事/领导提前打好招呼
```

### 合同注意事项

```
⚠️ 必须看：
1. 薪资构成（Base + 奖金）
2. 试用期时长（不超过6个月）
3. 试用期工资（≥80%正式工资）
4. 竞业协议（能否去竞品）
5. 保密协议（NDA）
6. 股权/期权协议

⚠️ 可以谈：
- 试用期不考核（争取无试用期）
- 竞业限制补偿金
- 签字费
```

## 3.8 常见问题 Q&A

```
Q：没有经验能进大厂吗？
A：可以，大厂有校招绿色通道，看潜力而非经验

Q：本科 vs 研究生？
A：大厂技术岗差距不大，研究生薪资略高 10-20%

Q：非计算机专业能入行吗？
A：可以，游戏行业更看重实际能力，项目经验很重要

Q：需要刷多少题？
A：目标大厂：200+（简单/中等为主）
   目标中厂：100+（简单为主）

Q：面试紧张怎么办？
A：多练，多 mock，紧张是正常的

Q：被拒了怎么办？
A：总结反馈，继续投，面试是一场概率游戏
```
