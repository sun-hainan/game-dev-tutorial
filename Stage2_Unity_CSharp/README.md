# Stage 2：Unity 2D/3D 开发

> 目标：掌握 Unity 引擎核心系统，独立完成 2D 游戏项目。

## 📖 章节列表

| # | 章节 | 核心内容 |
|---|------|---------|
| 01 | Unity 编辑器入门 | 界面布局，GameObject/Component，Transform |
| 02 | Unity 资源管理与项目结构 | 资源导入，Prefab，Scene 管理 |
| 03 | C# 核心语法对比 C++ | 语法差异，委托/事件，LINQ |
| 04 | Unity 脚本开发 MonoBehaviour | 生命周期函数，Update/FixedUpdate |
| 05 | Input 输入系统 | 键盘/鼠标/触摸，输入框架 |
| 06 | 2D 游戏开发基础 | Sprite/Tilemap，2D 物理，摄像机 |
| 07 | 物理引擎与碰撞检测 | Rigidbody，Collider，碰撞回调 |
| 08 | UI 系统 | Canvas/UGUI，Text/Button/Image，布局 |
| 09 | 协程与异步编程 | Coroutine，异步加载，Task |
| 10 | 预制体与对象池 | Prefab 实例化，对象池模式 |
| 11 | 场景管理与存档系统 | 场景切换，存档/读档，JSON 序列化 |
| 12 | 音频系统 | AudioSource，背景音乐，音效管理 |
| 13 | 动画系统 Animator | Animation Clip，Animator Controller，状态机 |
| 14 | 粒子系统 | Particle System，特效制作 |
| 15 | 完整 2D 游戏项目 | 从零完成平台跳跃游戏 |

## 🎯 学习目标

- 熟练使用 Unity 编辑器
- 掌握 C# 与 C++ 的核心差异（委托、事件、协程）
- 能够开发完整的 2D 游戏
- 理解游戏开发的完整流程：策划 → 原型 → 实现 → 打磨
