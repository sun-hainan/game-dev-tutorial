# Chapter 01：Unity 编辑器入门

> 🎯 目标：熟悉 Unity 编辑器界面、各窗口功能、学会创建项目和基础操作。

---

## 1.1 Unity 是什么？

### 生活中的类比 🏗️

> **Unity = 游戏世界的"万能乐高工厂"：**
> - 你不需要自己造塑料积木块（Unity 提供了现成的物理、渲染、音频模块）
> - 你只需要用代码把这些积木拼成你想要的游戏
> - 支持 2D、3D、AR、VR、手机、PC、主机全平台

### Unity 的核心特点

| 特点 | 说明 |
|-----|------|
| **跨平台** | 同一套代码发布到 25+ 平台 |
| **可视化编辑器** | 拖拽操作，直观所见即所得 |
| **组件化** | 游戏对象 = 积木块，每个块有不同功能 |
| **C#/JS 脚本** | 用代码控制游戏逻辑 |
| **资源商店** | 有大量付费/免费插件可用 |

## 1.2 Unity Hub 安装与项目创建

### 步骤 1：安装 Unity Hub

> 你已安装 Unity Hub 3.3.3 ✅

### 步骤 2：安装 Unity 编辑器版本

```
1. 打开 Unity Hub
2. 点击左侧 "安装"（Install）
3. 点击 "安装编辑器"
4. 推荐安装：Unity 2022.3 LTS（长期支持版，稳定）
5. 选择以下模块：
   ✅ WebGL Build Support（发布到网页）
   ✅ Windows Build Support（发布到 Windows）
   ✅ Android Build Support（发布到安卓）
   ✅ iOS Build Support（发布到苹果，需 Mac）
6. 点击 "安装"
```

### 步骤 3：创建第一个项目

```
1. Unity Hub → "项目"（Projects）
2. 点击 "新建项目"（New Project）
3. 选择模板：
   - 3D（默认）
   - 2D（做 2D 游戏选这个）
   - URP（通用渲染管线，推荐新项目用）
4. 项目名称：MyFirstGame
5. 存储位置：选择桌面或 D:\UnityProjects\
6. 点击 "创建项目"
```

## 1.3 Unity 编辑器界面详解

### 界面布局

```
┌─────────────────────────────────────────────────────────────┐
│                        顶部工具栏                           │
├──────────┬────────────────────────────┬────────────────────┤
│          │                            │                    │
│  层级    │                            │    检查器           │
│  窗口    │       场景视图              │    (Inspector)     │
│ (Hierarchy│      (Scene View)          │                    │
│          │                            │    - Transform     │
│          │                            │    - Collider      │
│          │                            │    - Renderer      │
│          ├────────────────────────────┤    - Script        │
│          │                            │                    │
│          │       游戏视图              │                    │
│          │      (Game View)           │                    │
├──────────┴────────────────────────────┴────────────────────┤
│                        项目窗口（Project）                    │
│              (所有资源文件：脚本/模型/图片/音频)               │
└─────────────────────────────────────────────────────────────┘
```

### 各窗口详解

#### 场景视图（Scene View）

> 你在游戏世界里的"摄像机"，用来布置场景。

```
操作：
- 鼠标右键 + WASD：自由飞行
- 鼠标右键 + Q/E：上下移动
- 鼠标滚轮：前进/后退
- F：聚焦选中对象
- 鼠标左键：选择对象
- 鼠标中键：平移视角
```

#### 游戏视图（Game View）

> 玩家看到的画面。点击"播放"按钮后运行游戏。

```
注意：播放时 Scene 视图和 Game 视图会同时显示
```

#### 层级窗口（Hierarchy）

> 当前场景中所有游戏对象的列表。

```
特点：
- 显示所有 GameObject（玩家、敌人、环境、UI 等）
- 有父子层级关系（父子节点）
- 支持搜索
```

#### 检查器窗口（Inspector）

> 选中对象的"属性面板"。

```
显示内容：
- Transform（位置/旋转/缩放）
- 所有组件（Mesh Renderer, Collider, Rigidbody...）
- 挂载的脚本及其 public 变量
```

#### 项目窗口（Project）

> 项目的所有资源文件，类似文件资源管理器。

```
常见文件夹：
Assets/         — 所有资源根目录
├─ Scenes/      — 场景文件
├─ Scripts/     — C# 脚本
├─ Prefabs/     — 预制体
├─ Materials/   — 材质
├─ Models/      — 3D 模型
├─ Sprites/     — 2D 图片
├─ Audio/       — 音频
└─ Scenes/      — 关卡场景
```

## 1.4 第一个 Unity 项目：3D 场景

### 创建地面

```
1. 在 Hierarchy 窗口点击右键
2. 3D Object → Plane（平面）
3. 在 Inspector 中设置：
   Transform:
     Position: (0, 0, 0)
     Scale: (5, 1, 5)    — 地面放大5倍
4. 给地面换个颜色：
   - 在 Project 窗口右键 → Create → Material
   - 命名为 GroundMaterial
   - 在 Material 的 Albedo 旁边点击圆圈，选一个绿色
   - 把 Material 拖到 Scene 里的 Plane 上
```

### 创建玩家方块

```
1. 在 Hierarchy 窗口点击右键
2. 3D Object → Cube
3. 设置位置：Transform Position: (0, 0.5, 0)（在地面上方）
4. 重命名：右键 → Rename → Player
5. 改颜色：创建 Material 命名为 PlayerMaterial，选蓝色，拖到 Cube 上
```

### 添加光源

```
1. 默认已有 Directional Light（平行光）
2. 可以调整方向让阴影更好看：
   - Transform Rotation: (50, -30, 0)
```

### 调整摄像机

```
1. Main Camera 默认在 (0, 1, -10)
2. 点击 Game 视图，可以看到玩家视角
3. 选中 Main Camera 后按 F 聚焦
```

### 保存场景

```
Ctrl + S → 保存当前场景
建议在 Scenes 文件夹下保存为 Scene1.unity
```

## 1.5 常用快捷键

| 快捷键 | 功能 |
|-------|------|
| `Ctrl + S` | 保存场景 |
| `Ctrl + Z` | 撤销 |
| `Ctrl + C/V` | 复制/粘贴对象 |
| `D` + 点击 | 复制选中对象 |
| `F` | 聚焦选中对象 |
| `Ctrl + Shift + F` | 将对象位置对齐到当前 Scene 视图 |
| `Q/W/E/R` | 工具：手形/移动/旋转/缩放 |
| `Space` | 在当前工具和 Pivot/Center 之间切换 |
| `Ctrl + D` | 复制对象（保留更多属性）|
| `Tab` | 在 Pivot 和 Center 之间切换 |
| `Alt + 鼠标左键拖动` | 旋转 Scene 视图 |
| `Shift + Alt + 鼠标左键` | 以选中对象为中心旋转 |

## 1.6 Unity 界面布局自定义

### 重置为默认布局

```
Layout → Default（顶部菜单栏）
```

### 保存自定义布局

```
1. 调整好你喜欢的布局
2. Layout → Save Layout...
3. 起个名字，比如"MyLayout"
```

### 推荐布局（2D 游戏）

```
Layout → 2 by 3（适合 2D 游戏）
```
