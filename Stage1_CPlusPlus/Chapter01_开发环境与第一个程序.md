## 1.3 第一个程序详解

### 完整代码

```cpp
// --------------------------------------------------
// 第一个 C++ 程序：Hello World
// 编译器：Visual Studio 2026 (MSVC)
// --------------------------------------------------

#include <iostream>     // 引入输入输出库
using namespace std;     // 使用标准命名空间

// -------- 主函数：程序入口 --------
int main()
{
    // -------- 输出语句 --------
    cout << "Hello, World!" << endl;   // 打印内容并换行

    // -------- 返回值 --------
    return 0;          // 程序正常结束
}
```

### 逐行解析

```
行号    代码                    解释
─────────────────────────────────────────────────────
 1     // --------...        注释：说明代码用途
 3     #include <iostream>    预处理器指令：将 iostream 文件内容复制到此处
                              // iostream = 输入输出流库，提供 cin/cout
 5     using namespace std;   使用标准命名空间，这样可以直接写 cout 而不用 std::cout
 8     int main()             主函数：程序执行的入口，必须返回 int（整数）
 9     {                      函数开始
10         cout << ...        cout：标准输出流对象，<< 表示把内容送进去
11                         endl：end line 的缩写，表示换行
12         return 0;           返回 0 表示程序正常退出（非0表示出错）
13     }                      函数结束
```

### 运行程序

```
在 Visual Studio 中：
  按 F5（调试运行）或 Ctrl + F5（不调试运行）

在命令行中（安装好 MSVC 编译器后）：
  cl /EHsc main.cpp      // 编译
  main.exe               // 运行
```

### 运行结果

```
Hello, World!
```

## 1.4 程序的基本结构

### 生活类比 🏠

```
程序结构          建筑类比
─────────────────────────────────
#include          建筑材料采购（引入工具）
using namespace   常用工具摆放位置
int main()        建筑物正门（必经入口）
{ ... }           建筑物内部空间
return 0          完工标记
```

### 核心语法规则

| 规则 | 说明 | 示例 |
|-----|------|------|
| 语句以 `;` 结尾 | 每个语句是一条命令 | `cout << "hi";` |
| 代码块用 `{}` | 包裹一段相关代码 | `int main() { ... }` |
| 注释用 `//` | 解释代码，不影响运行 | `// 这是注释` |
| 大小写敏感 | `cout` 和 `Cout` 不同 | 严格区分大小写 |

## 1.5 常见错误与解决

| 错误现象 | 常见原因 | 解决方法 |
|---------|---------|---------|
| `cout` 标红 | 缺少 `#include <iostream>` | 添加头文件 |
| `endl` 标红 | 缺少 `using namespace std;` | 添加这行 |
| 中文乱码 | 文件编码问题 | 保存为 UTF-8 或将中文改为英文 |
| 程序闪退 | 运行完自动关闭 | 在 `return 0;` 前加 `system("pause");` |
