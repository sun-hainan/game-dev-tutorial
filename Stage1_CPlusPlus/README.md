# Stage 1：C++ 基础

> 目标：掌握 C++ 核心语法，理解内存模型，为游戏开发打下坚实基础。

## 📖 章节列表

| # | 章节 | 核心内容 |
|---|------|---------|
| 01 | 开发环境与第一个程序 | VS2022 安装，main() 入口，程序结构 |

| 03 | 流程控制 | if/else, switch, for/while, break/continue |
| 04 | 函数与模块化 | 函数声明/定义，参数传递，递归 |

## 🎯 学习目标

- 理解 C++ 编译-链接-执行全流程
- 掌握指针和内存管理（游戏开发核心技能）
- 能够独立实现常用数据结构和算法
- 为 Stage 2 Unity/C# 开发做好准备
